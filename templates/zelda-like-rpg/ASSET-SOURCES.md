## Notes

- CraftPix assets require free account to download
- CraftPix direction convention: DOWN=0, LEFT=1, RIGHT=2, UP=3 (row order in sprite sheets)
- Always verify sprite sheet frame sizes and row layouts before writing animation code
- Some sprite sheets have non-uniform frame counts across directions
